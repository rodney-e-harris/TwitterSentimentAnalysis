Twitter Sentiment Analysis: A Python application that returns the sentiment analysis of the most recent tweets based on keyword entry!

Enter a keyword and it'll return 100 tweets relating to the keyword. This limit can be adjusted in app.py.

You can learn how to generate Twitter API keys at https://developer.twitter.com/en/docs/twitter-api/getting-started/getting-access-to-the-twitter-api

Because this is application is meant to be run in an isolated environment, the keys can be added in app.py for the purposes of testing.
